Chat with MySQL: AI-Powered SQL Assistant
<br>
An AI-driven application that allows you to interact with your MySQL database in natural language. This project leverages LangChain, Groq AI, and Streamlit to provide an intuitive interface for querying your database without needing in-depth SQL knowledge. Simply type your questions, and the AI will generate SQL queries, execute them on the connected database, and respond with the results in plain language.

Key Features:
Natural Language Querying: Ask questions about your database, and the AI generates the corresponding SQL queries.
Real-Time Database Interaction: Connect to your MySQL database with ease using environment variables for secure authentication.
Chat Interface: A user-friendly, conversational interface built with Streamlit to simulate a real-time chat experience.
Customizable: Easily adapt the prompt templates or extend functionality for other databases.
